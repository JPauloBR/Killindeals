

# KillinDeals
#### An app to retrieve and add notes on news 

## Synopsis

KillinDeals is an scraping app configured to scrape kotaku.com and retrieve most recent news.

## Technologies Used
* [express]
* [mongodb]
* [mongoose]
* [cherrio]
* [jquery]
* [ajax calls]
